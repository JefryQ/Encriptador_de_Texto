SECURICRYPT

SecuriCrypt es una herramienta que te permite encriptar y desencriptar texto a partir de un algoritmo de cifrado.

Instrucciones:

Para encriptar un texto simplemente hay que poner en el recuadro superior el texto que quieres encriptar y poner una contraseña, si lo deseas. Después de enviarlo desaparecerá el texto original del recuadro y aparecerá el texto cifrado disponible para copiar
